The main objective is to build a practical, secure, and scalable accounting application for companies that need daily financial tracking and invoice automation without relying entirely on manual spreadsheets.
•	Create a centralized system for managing company accounting records.
•	Implement double-entry bookkeeping so that every posted financial event is balanced.
•	Allow users to record income, expenses, vendor bills, customer invoices, and payments.
•	Automatically generate and email customer invoices once every month when recurring invoice rules are enabled.
•	Provide financial visibility through dashboards and reports such as Profit and Loss, Balance Sheet, Trial Balance, General Ledger, Accounts Receivable Aging, and Accounts Payable Aging.
•	Support basic tax/VAT/GST tracking according to configurable tax rates.
•	Allow secure user access, role-based permissions, audit logs, file attachments, and exportable data.
•	Deliver a working application within 2 months, with a 15-day grace period if needed under the agreed terms.
